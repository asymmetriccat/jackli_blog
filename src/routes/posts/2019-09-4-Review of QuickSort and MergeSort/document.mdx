**QuickSort**

The key of quicksort is using divide and conquer strategy. It picks an element as pivot and partitions the given array around the picked pivot. In the partition,
it uses two pointers low and high to scan from both sides each of time. If the value at high is smaller than pivot value, assign the high value to low value, otherwise
do the same from the low side, until low==high, return the index of low for the next round quick sort of the sub-arrays.


 Below is the code for quicksort which always picks the low value as the pivot:

**MergeSort**

MergeSort is also a divide and conquer algorithm. It recursively divides the array into halfs and do mergesort of the corresponding sub-arrays,
and merge the sorted sub-arrays.

Below is the code for MergeSort:

 ```java

 class MergeSort {
     public void mergesort(int [] a, int n){  // n is the length of array a
          if(n<2) return;
          int mid = n/2;
          int [] s1 =Arrays.copyOfRange(a, 0, mid);
          int [] s2 = Arrays.copyOfRange(a, mid, n);
          mergesort(s1, s1.length);
          mergesort(s2, s2.length);
          merge(a, s1, s2);
     }

     private void merge(int[] a, int [] s1, int [] s2){

                    i++;
               }
               else{
                   a[i+j] = s2[j];
                   j++;
               }
          }
     }
 ```
